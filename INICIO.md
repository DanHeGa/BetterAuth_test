# Material Para Estudiantes

## Laboratorio de Better Auth

Este paquete contiene todo lo necesario para realizar el laboratorio:

- guía paso a paso
- frontend en `Vite + React + TypeScript`
- backend en `Express + Better Auth`
- archivos de configuración de ejemplo

## Orden recomendado

1. Lee `GUIA_LABORATORIO.md`.
2. Revisa `README.md`.
3. Copia los archivos `.env.example` a `.env`.
4. Instala dependencias con `npm install`.
5. Ejecuta la migración con `npm run db:migrate`.
6. Opcionalmente crea el usuario demo con `npm run seed`.
7. Inicia el proyecto con `npm run dev`.

## Estructura

- `client/`: frontend del laboratorio
- `server/`: backend del laboratorio
- `GUIA_LABORATORIO.md`: explicación detallada
- `README.md`: resumen técnico del proyecto
